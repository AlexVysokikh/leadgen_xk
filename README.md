# 🎯 Кубик Медиа — Агент лидогенерации

Инструкция по запуску проекта на Windows.

## Шаг 1 — Скачать проект
1. Скачайте ZIP-архив репозитория через GitHub (кнопка **Code** → **Download ZIP**).
2. Распакуйте архив в удобную папку, например: `C:\Projects\leadgen_xk`

## Шаг 2 — Установить зависимости
1. Откройте папку проекта в редакторе **Cursor** (`File` → `Open Folder`).
2. Откройте терминал в Cursor (клавиши `Ctrl` + `~`).
3. Выполните команду для установки всех необходимых библиотек:
   ```bash
   pip install -r requirements.txt
   ```

## Шаг 3 — Создать файл .env
1. Скопируйте файл `.env.example` в корне проекта и переименуйте копию в `.env`.
2. Откройте `.env` и вставьте ваши реальные ключи:
   - `YANDEX_API_KEY` — ключ API Яндекс Карт (получить на [developer.tech.yandex.ru](https://developer.tech.yandex.ru))
   - `OPENAI_API_KEY` — ваш API ключ BotHub (получить на [bothub.chat](https://bothub.chat))
   - `OPENAI_BASE_URL` — `https://bothub.chat/api/v2/openai/v1`

## Шаг 4 — Запустить админку
В терминале Cursor выполните команду:
```bash
streamlit run main_app.py
```
После запуска браузер откроется автоматически по адресу: [http://localhost:8501](http://localhost:8501)

## Шаг 5 — Первый запуск по порядку
Для проверки работоспособности выполните действия в следующей последовательности:
1. **Раздел "🔍 Поиск"**: Выберите район Москвы → Введите тип бизнеса (например, "кафе") → Нажмите кнопку "Найти компании".
2. **Раздел "📋 База лидов"**: Убедитесь, что найденные компании появились в таблице.
3. **Раздел "⚙️ Управление"**: Нажмите кнопку "🌐 Собрать контакты с сайтов".
4. **Раздел "⚙️ Управление"**: Нажмите кнопку "🤖 Сгенерировать КП".
5. **Раздел "✏️ Модерация КП"**: Проверьте сгенерированные тексты в карточках и нажмите "✅ Одобрить".
6. **Раздел "⚙️ Управление"**: Нажмите кнопку "📤 Симулировать рассылку".

🏁 Проект готов к работе!
